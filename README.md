# 🛒 Market Hafıza Oyunu

Yaşlı teyzelerin market fiyat kontrolü macerasını oyuna çevirdik!  
Reyonlarda ürünleri gez, fiyatları aklında tut, kasada fişteki hatayı bul!

## Nasıl Oynanır?
- WASD / Joystick ile reyonlarda gez  
- Ürünlere tıkla → fiyatı gör + aklına yaz  
- 6 ürün dolunca kasaya git ve fişteki yanlış fiyatı bul!

## Geliştirme
- Unity 2022.3+  
- 2D Top-Down  
- AdMob reklam entegrasyonu gelecek
